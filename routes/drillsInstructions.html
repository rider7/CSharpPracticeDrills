Practice your C# skills by building this quick console application!

  • This app is uses the same calculator logic we discussed on Wednesday from the curriculum.
    Check the gitbook week 1 for a review of the logic.
  • This app will be used to find the sum of two numbers and must meet the following parameters:
    • Console should write the text "Addition Time!"
    • Console should write the text "Press Enter To Begin"
    • User should then have to press enter to proceed
    • Console should write the text "What is your first number?"
    • User should then be able to enter the first number.
    • Console should write the text "What is your second number?"
    • User should then be able to enter the second number.
    • Console should write the "Here is your answer!"
    • Console should write the sum of the two numbers showing the full expression in the format below. Example: 3 + 5 = 8
  • For more practice create this app for subtraction, multiplication, and division!
